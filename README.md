# mcp-bls

BLS MCP — Bureau of Labor Statistics public data API (v2)

Part of the [Pipeworx](https://pipeworx.io) open MCP gateway.

| Tool | Description |
|------|-------------|

## Quick Start

Add to your MCP client config:

```json
{
  "mcpServers": {
    "bls": {
      "url": "https://gateway.pipeworx.io/bls/mcp"
    }
  }
}
```

Or use the CLI:

```bash
npx pipeworx use bls
```
